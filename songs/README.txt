Place your song files in this folder!

Expected files for the default setup:
- track1.mp3
- track2.mp3
- track3.mp3
- track4.mp3

You can add more songs and update the music panel in index.html to include them.

Supported formats: .mp3, .wav, .ogg
Recommended: MP3 format at 192-320 kbps for best web compatibility
